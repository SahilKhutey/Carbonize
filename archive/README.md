# CMBSG (Continuous Multiphase Biomineralization & Solidification Grid) - Archive

This directory contains the legacy, standalone "CMBSG" implementation (pip + venv + docker-compose).

It was superseded by the core **CBMS-Sim** monorepo workspace design inside the `packages/` directory, which utilizes Poetry and pnpm workspaces.

This legacy code has been moved here to:
1. Prevent onboarding confusion for new developers.
2. Clean up the repository root directory.
3. Maintain the historical implementation details for reference if needed.
