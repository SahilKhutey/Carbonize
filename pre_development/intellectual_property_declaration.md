## 2. Protected Scope & Covered Assets

This ownership declaration covers the following materials:

*   **sim-core domain engine:** All JIT-compiled BDF kinetics solvers, Monte Carlo Latin Hypercube Sampling pipelines, Spearman rank Sobol sensitivity indices, First Passage Time solvers, compressive block strength classifiers, and financial discount models.
*   **FastAPI backend app:** All API routing structures, middlewares, optimistic locking schemas, JWT tenant authentication controls, and Redis event brokers.

*   Specific slurry formulation designs, operational condition parameter intervals, and curing configurations.
